Redux - NgRx

Action

Objeto: Única fuente de información que se envía por interacciones de usuario o programa
Lo más simples posible

  • type: constante, obligatorio
  • payload: opcional. Menor cantidad de información para realizar una acción

Reducer

Función pura que recibe el Action y el State
Retorna un State

  • oldState: Estaedo actual ---> Inmutable
  • Action: Objeto plano que indica qué hay que hacer

State

Objeto con informaición transversal a toda la app. Sólo lectura. No se mutará de forma directa
Prohibido: Push o manipulación directa

Store

Permite la lectura del estado via getStsate()
Permite crear un nuevo estado via dispatc(Action) Permite notificar cambios via Subscribe()

Effects

Escucha a las acciones despachadas por el el ngrx-Store
Simplifica la lógica en los coponentes
Comunicación fuera de Angular: HTTP, sockets, tareas asíncronas

  • Se comunica con un servicio
  • Escucha el servicio
  • Dispara una nueva acción
