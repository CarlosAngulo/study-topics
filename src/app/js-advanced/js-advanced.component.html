Closures

  • Funciones que manejan variables independientes
  • Tipo especial de objeto que combina dos cosas: una función, y el entorno en que se creó esa función
  • Permite asociar algunos datos (el entorno) con una función que opera sobre esos datos.
  • Creación de métodos privados al estilo de OOP.

  {{
    'function creaSumador(x) {
      return function(y) {
        return x + y;
      };
    }
    
    var suma5 = creaSumador(5);
    var suma10 = creaSumador(10);
    
    console.log(suma5(2));  // muestra 7
    console.log(suma10(2)); // muestra 12'
  }}

Authentication
carlosazaustre.es
  • Servidor, almacenando la sesión
  • Autenticación sin estado con Tokens
    No hay información de estado y la aplicación se vuelve totalmente escalable. Al no utilizar cookies para almacenar la información del usuario, podemos evitar ataques CSRF (Cross-Site Request Forgery)
  • JWT: Autenticación con JSON Web Tokens

    Header: La primera parte es la cabecera del token, que a su vez tiene otras dos partes, el tipo, en este caso un JWT y la codificación utilizada.

    Payload: EL Payload está compuesto por los llamados JWT Claims donde irán colocados la atributos que definen nuestro token.

    Signature: Está formada por los anteriores componentes (Header y Payload) cifrados en Base64 con una clave secreta (almacenada en nuestro backend).
