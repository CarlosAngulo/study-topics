ANGULAR

- Framework: includes a set of functionalities builted in a package > Directives, serivces, pipes, etc, but also third part libraries: typescript, rxjs

- Platform: Has a rich ecosystem Karma, Jazmin, Angular CLI, Angular Playground, Angula Universal, TestBed, Material.

      '@NgModule({

      })' 
    }}

Definition

  • Class, Artefact, Namespace -> Groups related items together
  • Directives, components, serivces, more modules, pipes
  • Divide the app into different domains
  • The basic building blocks of an Angular application are NgModules
  • Los NgModules son contenedores para un bloque cohesivo de código dedicado a un dominio de aplicación, un flujo de trabajo o un conjunto de capacidades estrechamente relacionadas.

Directivas

  
    {{
      '@Directive({
        selector: [ appHighlight ]
      })' 
    }}
  1. Components—directives with a template. - Container para piezas de negocio. Interfaz - Directiva que tiene asignado un template
  2. Structural directives—change the DOM layout by adding and removing DOM elements. NgIf, NgFor, ngSwitch
  3. Attribute directives—change the appearance or behavior of an element, component, or another directive. ngClass, NgStyle

Definition

  • Se usan para presentación y problemas estructurales.
  • Atributo o marcador en una etiqueta HTML, este atributo está dotando al elemento del DOM que la contiene de un comportamiento, ese comportamiento lo definimos nosotros y se puede utilizar en todos los elementos que queramos.

Componentes

  
    {{
      '@Component({
        selector: \'app-component\'
        templateUrl: \'./app-component.html\',
        styleUrls: [\'./app-component.scss\'],
        providers:  [ HeroService ]
      })' 
    }}

Definition

  • Is a directive with a template. It is composed of HTML, CSS, Logic through TS file and a spec. Minimum logic unit in angular.

Pipes

  
    {{
      '@Pipe({ name: \'pipeName\' })
      export class ExponentialStrengthPipe implements PipeTransform {
        transform(value: number, exponent: string): number {...}' 
    }}

Definition

  • A class with a transformation function for the HTML. UPPERCASE. For the view only
  • Asinc: Pipe that allows you to subscribe in the view to an observable. It unsubscribes as soon as the observable completes. (uppon completion)
